## Logo Guidelines

### Primary Logo

The Interact logo consists of:
- Wordmark: "Interact"
- Icon: (To be designed)

### Logo Variations

**Primary Logo** - Full color on light backgrounds  
**Reversed Logo** - White on dark backgrounds  
**Monochrome Logo** - Single color version  
**Icon Only** - For small spaces (app icons, favicons)

### Clear Space

Maintain a minimum clear space around the logo equal to the height of the letter "I" in "Interact" on all sides.

### Minimum Size

- **Digital:** 120px width minimum
- **Print:** 0.75 inches width minimum
- **Favicon:** 32x32px, 16x16px

### Logo Don'ts

❌ Don't stretch or distort the logo  
❌ Don't change logo colors  
❌ Don't rotate the logo  
❌ Don't add effects (shadows, gradients)  
❌ Don't place on busy backgrounds  
❌ Don't recreate or modify the logo

## Typography

### Font Families

**Primary Font: Inter**
```css
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', 'Roboto', sans-serif;
```

**Monospace (Code): JetBrains Mono**
```css
font-family: 'JetBrains Mono', 'Consolas', 'Monaco', monospace;
```

### Type Scale

```css
/* Headings */
--text-6xl: 3.75rem;    /* 60px */
--text-5xl: 3rem;       /* 48px */
--text-4xl: 2.25rem;    /* 36px */
--text-3xl: 1.875rem;   /* 30px */
--text-2xl: 1.5rem;     /* 24px */
--text-xl: 1.25rem;     /* 20px */
--text-lg: 1.125rem;    /* 18px */

/* Body */
--text-base: 1rem;      /* 16px */
--text-sm: 0.875rem;    /* 14px */
--text-xs: 0.75rem;     /* 12px */
```

### Font Weights

```css
--font-light: 300;
--font-normal: 400;
--font-medium: 500;
--font-semibold: 600;
--font-bold: 700;
--font-extrabold: 800;
```

### Line Heights

```css
--leading-none: 1;
--leading-tight: 1.25;
--leading-snug: 1.375;
--leading-normal: 1.5;
--leading-relaxed: 1.625;
--leading-loose: 2;
```

### Typography Guidelines

- **Headings:** Use semibold (600) or bold (700)
- **Body Text:** Use normal (400) or medium (500)
- **Small Text:** Use normal (400)
- **Buttons:** Use medium (500) or semibold (600)
- **Line Height:** 1.5 for body, 1.2 for headings

## Imagery

### Photography Style

- **Authentic** - Real people in real workplace settings
- **Diverse** - Represent all demographics and roles
- **Energetic** - Show engagement and connection
- **Professional** - High quality, well-lit images
- **Candid** - Natural moments, not overly posed

### Illustrations

- **Style:** Modern, friendly, geometric
- **Colors:** Brand color palette
- **Usage:** Complement, not replace, content
- **Consistency:** Use same illustration style throughout

### Icons

**Primary:** Lucide Icons
- Consistent stroke width
- 24x24px default size
- Use brand colors or neutrals

### Image Guidelines

✅ **Do:**
- Use high-resolution images (2x for retina)
- Optimize for web (WebP format preferred)
- Include alt text for accessibility
- Maintain consistent aspect ratios

❌ **Don't:**
- Use generic stock photos
- Apply heavy filters or effects
- Use low-quality or pixelated images
- Stretch or distort images

## Voice & Tone

### Brand Voice

**Characteristics:**
- **Professional** - Expert and credible
- **Approachable** - Friendly and accessible
- **Encouraging** - Positive and motivating
- **Clear** - Simple and direct

### Tone Variations

**Marketing/Landing Pages:**
- Enthusiastic and inspiring
- Highlight benefits and value
- Use active voice
- Include calls-to-action

**Product/UI:**
- Clear and concise
- Action-oriented
- Helpful and instructive
- Consistent terminology

**Documentation:**
- Professional and thorough
- Step-by-step and detailed
- Use examples and code snippets
- Technical accuracy

**Support/Help:**
- Empathetic and patient
- Problem-solving focused
- Reassuring and helpful
- Solution-oriented

### Writing Guidelines

✅ **Do:**
- Use active voice: "Create an activity" not "An activity should be created"
- Be concise: Remove unnecessary words
- Use "you" to address users
- Include specific examples
- Use bullet points for lists

❌ **Don't:**
- Use jargon without explanation
- Write long, complex sentences
- Use passive voice
- Be overly casual or informal
- Include placeholder text in production

## Usage Guidelines

### Web & Digital

**Website/Application:**
- Follow responsive design principles
- Maintain consistent spacing (8px grid system)
- Use elevation (shadows) consistently
- Ensure WCAG 2.1 AA accessibility

**Email:**
- Use web-safe fonts as fallbacks
- Optimize images for email clients
- Test across devices and clients
- Include plain text version

**Social Media:**
- Adapt logo for profile pictures
- Use brand colors in graphics
- Maintain consistent visual style
- Include alt text for accessibility

### Print Materials

**Business Cards:**
- Logo on front
- Contact information clearly legible
- Use brand colors
- High-quality printing (300 DPI minimum)

**Presentations:**
- Use brand slide template
- Consistent header/footer
- Brand colors and fonts
- High-quality images

## Accessibility

### Color Contrast

All text must meet WCAG 2.1 AA standards:
- **Normal text:** 4.5:1 contrast ratio
- **Large text (18px+):** 3:1 contrast ratio
- **UI components:** 3:1 contrast ratio

### Color Blindness

- Don't rely on color alone to convey information
- Use patterns, icons, or text labels as well
- Test designs with color blindness simulators
